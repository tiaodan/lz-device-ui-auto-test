# 项目说明

## 项目概述

无人机防御系统 UI 自动化测试工具，支持三种测试类型：

| 测试类型 | 说明 | 技术 |
|---------|------|-----|
| **断言测试** (assertion_test) | 验证元素存在、可见、文本正确 | Playwright expect |
| **视觉测试** (visual_test) | 截图对比，检测UI变化 | Pillow 图片对比 |
| **API测试** (api_test) | GraphQL接口验证 | requests + GraphQL |

## 技术栈

- **测试框架**: pytest + pytest-playwright
- **日志框架**: loguru
- **浏览器自动化**: Playwright
- **图片处理**: Pillow
- **配置管理**: YAML

## 配置文件 (config/test_config.yaml)

```yaml
# 环境配置
base_url: "https://192.168.85.238"
username: "root"
password: "lz"
ignore_ssl_errors: true

# 浏览器配置
browser:
  type: chromium
  headless: true

# 日志配置
log_path: logs
log_level: INFO

# 视觉测试配置
visual:
  baseline_path: screenshots/baseline
  current_path: screenshots/current
  diff_path: screenshots/diff
  threshold: 0.1  # 差异阈值10%
  auto_update_baseline: false

# 断言测试配置
assertion:
  timeout: 5000
  screenshot_on_failure: true
```
