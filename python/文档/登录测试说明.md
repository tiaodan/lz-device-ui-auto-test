# 登录测试说明

## 登录页面元素定位

根据实际页面（https://192.168.85.238）检测：

| 元素 | 定位器 | HTML结构 |
|-----|--------|----------|
| 用户名输入框 | `#username` | `<input id="username">` |
| 密码输入框 | `#password` | `<input id="password">` |
| 登录按钮 | `.lBtn` | `<button class="... lBtn">Sign in</button>` |
| 滑块容器 | `.drag` | `<div class="drag">...</div>` |
| 滑块按钮 | `.drag .btn` | `<div class="btn">` |
| 滑块进度 | `.drag .progress` | `<div class="progress">` |
| 滑块文本 | `.drag .text` | `<span class="text">Please slide to unlock</span>` |

## 登录流程

1. 填写用户名密码
2. 拖动滑块验证码到右侧
3. 点击登录按钮

## 滑块验证码处理

滑块 HTML 结构：
```html
<div class="drag">
  <div class="bg">
    <span class="text">Please slide to unlock</span>
  </div>
  <div class="progress" style="width: 0px;"></div>
  <div class="btn" style="left: 0px;"></div>
</div>
```

JavaScript 模拟拖动：
```python
def drag_slider(self):
    """拖动滑块"""
    page.evaluate("""() => {
        const btn = document.querySelector('.drag .btn');
        const drag = document.querySelector('.drag');
        const targetLeft = drag.offsetWidth - btn.offsetWidth - 10;
        
        btn.style.left = targetLeft + 'px';
        btn.dispatchEvent(new MouseEvent('mousedown', { bubbles: true }));
        document.dispatchEvent(new MouseEvent('mousemove', { bubbles: true }));
        document.dispatchEvent(new MouseEvent('mouseup', { bubbles: true }));
    }""")
```
