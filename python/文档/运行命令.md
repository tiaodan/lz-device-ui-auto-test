# 运行命令

## pytest 基本命令

```bash
cd python

# 运行全部测试
pytest

# 详细输出
pytest -v

# 显示打印日志
pytest -s

# 运行指定文件
pytest assertion_test/test_login.py

# 运行指定测试
pytest assertion_test/test_login.py::test_login_page_elements

# 显示浏览器（非无头模式）
pytest --headed

# 指定浏览器
pytest --browser chromium
pytest --browser firefox
```

## 按测试类型运行

```bash
# 断言测试
pytest -m assertion

# 视觉测试
pytest -m visual

# API测试
pytest -m api

# 登录相关测试
pytest -m login

# 排除慢速测试
pytest -m "not slow"

# 组合条件
pytest -m "assertion and login"
pytest -m "assertion or visual"
```

## 并行执行

```bash
# 自动检测CPU核心数并行
pytest -n auto

# 指定进程数
pytest -n 4

# 按测试类型并行
pytest -n auto -m assertion
```

## 失败重试

```bash
# 失败重试2次
pytest --reruns 2

# 重试间隔1秒
pytest --reruns 2 --reruns-delay 1
```

## 测试报告

```bash
# HTML报告
pytest --html=reports/report.html --self-contained-html

# Allure报告
pytest --alluredir=allure-results
allure serve allure-results

# 打开HTML报告
start reports/report.html
```

## 日志控制

```bash
# 控制台日志级别
pytest --log-cli-level=DEBUG

# 禁用控制台日志
pytest --log-cli-level=WARNING

# 查看详细日志
pytest -s --log-cli-level=INFO
```

## 测试发现

```bash
# 查看会收集哪些测试
pytest --collect-only

# 查看测试数量
pytest --collect-only -q
```

## 失败处理

```bash
# 首次失败即停止
pytest -x

# 失败2次后停止
pytest --maxfail=2

# 只运行失败的测试
pytest --lf

# 先运行失败的测试
pytest --ff
```

## 调试

```bash
# 进入pdb调试（失败时）
pytest --pdb

# 进入pdb调试（开始时）
pytest --trace

# 详细错误信息
pytest --tb=long

# 简短错误信息
pytest --tb=short

# 无错误追踪
pytest --tb=no
```

## 常用组合命令

```bash
# 快速运行（并行+简短错误）
pytest -n auto --tb=short

# 完整验证（报告+Allure）
pytest -v --html=reports/report.html --alluredir=allure-results

# CI环境常用
pytest -v -n auto --tb=short --html=reports/report.html --self-contained-html

# 调试模式
pytest -s --headed --tb=long -x

# 只跑API测试（无浏览器）
pytest -n auto -m api
```

## 测试结果

测试通过后：
- 日志保存在 `logs/` 目录
- 截图保存在 `screenshots/` 目录
- HTML报告保存在 `reports/` 目录

测试失败后：
- 失败截图保存在 `screenshots/failures/`
- 差异图保存在 `screenshots/diff/`
