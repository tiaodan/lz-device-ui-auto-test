使用方式

# 1. 安装依赖
pip install -r requirements.txt
playwright install

# 2. 编辑配置（修改用户名密码）
vim config/test_config.yaml

# 3. 运行测试
python run_tests.py              # 运行所有
python run_tests.py assertion    # 只运行断言测试
python run_tests.py visual       # 只运行图片识别
python run_tests.py api          # 只运行接口测试

# 4. 创建基准图（首次）
python run_tests.py visual baseline



修改配置，把 headless: false，你就能看到浏览器操作过程


单独运行命令-测某个模块：

python run_tests.py api - 运行所有 API 测试
python run_tests.py api login - 运行所有 login 测试
python run_tests.py api login_ok - 只运行正常登录测试
python run_tests.py api login_error_normal - 只运行异常通用测试
python run_tests.py api login_error_business - 只运行异常业务测试


测试类型	状态	说明
断言测试	✅ 通过	页面元素、滑块进度验证
图片识别	✅ 运行	基准图已创建
接口测试	📝 待验证	需配置正确的 GraphQL URL
