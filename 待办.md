UI异常模拟器说明
原理：JavaScript注入
不需要逆向破解，通过 Playwright 的 page.evaluate() 直接注入 JavaScript 代码，修改前端 DOM/CSS：


# 示例：隐藏元素
page.evaluate("""
    () => {
        const el = document.querySelector('.lBtn');
        if (el) el.style.display = 'none';
    }
""")
支持的UI异常类型
异常类型	方法	效果
元素隐藏	hide_element(selector)	元素消失，display:none
样式异常	break_element_style(selector)	红色背景、蓝色文字、5px字体
位置错乱	break_element_position(selector)	absolute定位、随机偏移、旋转
按钮禁用	disable_button(selector)	disabled + opacity:0.5
输入框失效	break_input(selector)	readOnly + 背景灰色
滑块失效	break_slider(selector)	pointerEvents:none + opacity:0.3
文字溢出	overflow_text(selector)	宽度50px + 长文本溢出
图片失败	break_image(selector)	替换为红色错误SVG
错误遮罩	inject_error_overlay(message)	全屏红色遮罩
移动端视图	simulate_mobile_view()	375px宽度布局错乱
